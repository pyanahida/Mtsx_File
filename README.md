# MT管理器 语法文件合集 | MT Manager Syntax Collection

本仓库为 [MT管理器](https://mt2.cn/) 提供多个编程语言的语法高亮支持（`.mtsx` 格式）。  
This repository provides syntax highlighting (`.mtsx` format) for [MT Manager](https://mt2.cn/).

---

## 中文

### 支持的语言

| 语言 | 文件 | 预览 |
|------|------|------|
| ArkTS | `ArkTS.mtsx` | [预览](ArkTS.mtsx_preview) |
| F# | `FSharp.mtsx` | [预览](FSharp.mtsx_preview) |
| Tcl | `Tcl.mtsx` | [预览](Tcl.mtsx_preview) |

### 安装方法

1. 下载本仓库中的 `.mtsx` 文件（如 `ArkTS.mtsx`）
2. 将文件放入手机存储的以下目录：  
   `/MT2/syntax/`
3. 打开 MT 管理器，在编辑器中打开对应类型的文件，语法高亮将自动生效

> 若目录不存在，请手动创建。

### 特性

- **ArkTS**：支持 HarmonyOS 应用开发语法  
- **F#**：支持函数式编程语法  
- **Tcl**：支持 Tcl/Tk 脚本语法

各语言的具体高亮细节，可参考对应的 `.mtsx_preview` 预览文件。

- [MT管理器](https://mt2.cn/) 提供的强大文件管理及编辑功能  
- 所有测试和反馈的朋友

## English

### Supported Languages

| Language | File | Preview |
|----------|------|---------|
| ArkTS | `ArkTS.mtsx` | [Preview](ArkTS.mtsx_preview) |
| F# | `FSharp.mtsx` | [Preview](FSharp.mtsx_preview) |
| Tcl | `Tcl.mtsx` | [Preview](Tcl.mtsx_preview) |

### Installation

1. Download the `.mtsx` files (e.g., `ArkTS.mtsx`) from this repository  
2. Place them into the following directory on your phone:  
   `/MT2/syntax/`
3. Open MT Manager and edit a file of the corresponding type; syntax highlighting will be applied automatically

> Create the directory manually if it does not exist.

### Features

- **ArkTS**: Syntax for HarmonyOS application development  
- **F#**: Functional programming syntax  
- **Tcl**: Tcl/Tk scripting syntax

For detailed highlighting rules, refer to the corresponding `.mtsx_preview` preview files.
